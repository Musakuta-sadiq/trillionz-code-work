Mocktails by Aabuu - Order System
===================================

HOW TO SET UP:
1. Save all three files (index.html, style.css, script.js) in the same folder
2. Open index.html in any web browser
3. The system is ready to use!

FEATURES:
- 4-step ordering process
- Dynamic order calculations
- Data validation
- Copy to clipboard functionality
- Local storage backup
- Mobile responsive design

TO CONNECT TO MICROSOFT FORMS:
1. Create a Microsoft Form for order collection
2. In script.js, find: const MICROSOFT_FORMS_LINK = "..."
3. Replace with your actual Microsoft Forms URL
4. Save and refresh the page

NOTE: This system works 100% offline in the browser
